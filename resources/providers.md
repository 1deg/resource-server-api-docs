## Providers

### `GET providers`

Returns a list of providers.

### `POST providers`

Creates a new provider.

### `GET providers/:id`

Returns a specific provider, along with the following nested resources:

- `attributes`
- `tags`
- `ratings`
- the primary `location`

### `PUT providers/:id`

Updates a provider.

### `DELETE providers/:id`

Deletes a provider.
